INSTRUCTIONS:

Libraries required:
libglfw
freetype
SOIL

Use make to compile, then execute adventure.

GAMEPLAY:

ARROW KEYS move the player in set directions.
R resets the game if you haven't won.
Q cycles through cameras.
F sets player to fast speed.
G sets player to slow speed.
WASD moves the camera in helicopter mode.

Reach the blue block and press R to win!
